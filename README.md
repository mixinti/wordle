# Wordle
# El juego tiene como objetivo adivinar la palabra secreta en un numero limitado de intentos, Despues de cada intento, el sistema del juego indica:

# 1. Letras correctas en la posicion correcta.
# 2. Letras correctas en posicion incorrecta.
# 3. Letras incorrectas.

# El jugador gana si adivina la palabra antes de quedarse sin intentos.
